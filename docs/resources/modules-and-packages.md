# Module kpeg library

Welcome to the **Kotlin PEG parser** with **Kotlin DSL**!

The project is inspired by the [pest parser](https://pest.rs/) and the [kotlin-peg-dsl project](https://github.com/mikaelhg/kotlin-peg-dsl).
